Class

TS class differs from JS in access modifiers checked only during compile-time: 

class Person {
  private name: String;
  private surname: String;
  
  
  constructor(name: String, surname: String) {
    this.name = name; //Must be initialized in the contructor!
    this.surname = surname;
  }

  public introduceHimself() {
    alert(`Hi! I am ${this.name} ${this.surname}`)
  }

  meetWith(withPerson: Person) {
    withPerson.introduceHimself();
  }
}

In TS, classes can be abstract.
