Modules

In Typescript, modules syntax depends on underlying JS system.

CommonJS

Historically first approach when modules syntax hasn't been yet introduced as ECMA standard. 

export.method = () => {...};


const someModule = require('some-module');
someModule.method();

Implementation of this pattern is a standard for Node.js: 

modules.exports.method = () => {...};

const someModule = require('some-module');
someModule.method();

AMD

Asynchronous Module Definition -introduced because asynchronous module loading was better suited for web browsers. 

export.method = () => {...};


const someModule = define('my-module', ['some-module'], function (someModule) {
  someModule.method();
});

One of implementation is for instance: RequireJS

ES6

First native implementation. 

export method = () => {...};


import { method } from 'some-module';
method();
